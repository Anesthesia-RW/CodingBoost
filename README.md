# CodingBoost



[toc]



## Introduction

CodingBoost 用于文献分享小组提高编码能力。



## Plan & Limit

* 每周一次，每次一题；
* 题目难度中等偏上；
* 仅可使用C语言(C99)。



## Usage

* 每周会新建一个`2021XXXX`文件夹，文件夹名为每周周一对应的日期；

* 每周的文件夹下每个人有一个文件夹，每个文件夹下对应每个人的编码；

  ```shell
  |-- CodingBoost
  |   |-- 2021XXXX
  |       |-- mucheng
  |           |-- main.c  // Source Code
  |           |-- docs.md  // Docs
  |       |-- renwei
  |           |-- main.c
  |           |-- docs.md
  |       |-- ningyimin
  |           |-- main.c
  |           |-- docs.md
  ```

* 每周的文件夹可以将`template`文件夹直接拷贝过来。
